Woocommerce Instance Connector v14
==================================
Module used for connecting woocommerce instance with
odoo in here woocommerce use as frontend and odoo as
backend.

Installation
============
	- www.odoo.com/documentation/14.0/setup/install.html
	- Install our custom addon

Configuration
=============

Instance need to be create for the further usage
Woocommerce --> instance.
